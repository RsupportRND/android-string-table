# android-string-table

## Introduction
- .xlsx 파일의 string 내용들을 res/values/strings_generated.xml 으로 변환한다.
- Language code 지원



## lib module 
Generate strings_generated.xml from .xlsx file

## download module 
Download .xlsx from google spreadsheet
